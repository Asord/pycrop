# PyCrop

Crop videos using ffmpeg.

## How to

To crop a video you have execute the command "python3 app.py <path to video>" or drag and drop the video onto the app.py file.

Use the slider to change the video timestamp

Select the area of the preview where you wish to crop the video using your mouse

Use the "Get Cropped Preview" to see the result on a temporary image

Use the "Get Preview" button to update the preview (note: preview is updated when the slider changes, this button is now useless)

Use the "Save" button to create the cropped video.

## FFMPEG

This program needs ffmpeg in order to work. You can update your ffmpeg installation folder by editing the line in tools.py
